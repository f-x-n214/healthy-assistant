## 三、异常检测/联动规则

### 3.1 L2：异常检测与主动通知

| 规则ID | 检测条件 | 告警级别 | 通知内容 |
| --- | --- | --- | --- |
| W1 | 温度 ≥ 38°C | ⚠️ 危险 | 高温红色预警！当前温度{temp}°C，请立即停止户外活动，保持室内通风降温 |
| W2 | 温度 ≥ 35°C | ⚠️ 警告 | 高温预警！当前温度{temp}°C，请减少户外活动时间 |
| W3 | 温度 ≤ 5°C | ⚠️ 警告 | 低温预警！当前温度{temp}°C，请务必注意保暖 |
| W4 | 风力 ≥ 6级 | ⚠️ 警告 | 大风预警！当前风力{windScale}级，请避免外出 |
| W5 | 天气 = 大雨/暴雨 | ⚠️ 危险 | 暴雨预警！{weather}，请避免外出，注意防涝 |
| W6 | 紫外线指数 ≥ 8 | ⚠️ 警告 | 紫外线强！UV指数{uvIndex}，外出请做好防晒措施 |

### 3.2 L3：数据驱动的建议调整

| 规则ID | 触发条件 | 智能响应 | 建议调整 |
| --- | --- | --- | --- |
| A1 | 温度 ≥ 35°C | 自动调整运动建议 | 建议避免中午户外活动，可在清晨或傍晚凉爽时散步 |
| A2 | 温度 ≤ 15°C | 自动调整运动建议 | 建议减少户外活动，注意保暖，可进行室内轻度运动 |
| A3 | 湿度 ≥ 85% | 自动调整健康建议 | 提醒用户注意防潮，建议使用除湿设备 |
| A4 | 湿度 ≤ 30% | 自动调整健康建议 | 提醒用户多喝水，建议使用加湿器 |
| A5 | 天气含"雨" | 自动调整运动建议 | 建议室内活动或在家休息 |
| A6 | 紫外线指数 ≥ 7 | 自动调整外出建议 | 提醒用户做好防晒措施 |

### 3.3 联动规则示例

**场景1：高温天气下的健康建议联动**
```
温度 ≥ 35°C → 检测到高温异常 → 
  ├─ 发送高温告警通知

```

**场景2：雨天的活动建议调整**
```
天气 = 雨天 → 检测到降水 → 
  ├─ 发送天气提醒

  └─ 提醒携带雨具
```
