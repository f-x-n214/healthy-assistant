# 用户调研原始材料

> 银发健康助手 · 课程论文附录 C  
> 调研要求：≥3 名真实目标用户（非同学互评）

## 文件说明

| 文件 | 用途 |
|------|------|
| `访谈提纲.md` | 访谈前阅读，统一提问 |
| `访谈记录_用户A.md` | 老人端被试 A 访谈与任务记录（填写后归档） |
| `访谈记录_用户B.md` | 老人端被试 B 访谈与任务记录（填写后归档） |
| `访谈记录_用户C.md` | 子女/照护者被试 C 访谈与任务记录（填写后归档） |
| `可用性测试表.md` | 3 项核心任务测试记录表 |
| `问卷.md` | 满意度问卷（可打印或微信发送） |
| `问卷结果汇总.md` | 3 人问卷汇总（调研后填写） |

## 调研流程（每人约 25–35 分钟）

1. 说明来意，征得同意（可录音）
2. 按 `可用性测试表.md` 完成 T1–T3（先观察，少提示）
3. 用户 C 额外完成 T4 子女端任务
4. 按 `访谈提纲.md` 半结构化访谈
5. 请用户填写 `问卷.md`（或你代为勾选）
6. 当天整理进各 `访谈记录_用户X.md`

## 合格调研对象

- ✅ 父母、祖父母、邻居老人、社区老人
- ✅ 子女、配偶等真实照护者
- ❌ 同班同学互评、室友扮演老人
