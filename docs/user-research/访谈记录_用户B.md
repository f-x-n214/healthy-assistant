## 访谈要点记录

### 1. 日常健康记录习惯

血糖用社区发的记录表，血压靠记忆，经常搞不清上周测了几次。吃药靠闹钟，周末孙子来会帮忙确认。

### 2. 最好用的功能

记完血压马上告诉「略偏高、注意休息」，听得懂；语音念例子跟着说，比打字省力。

### 3. 最难用 / 看不懂的地方

眼睛花，输入框字还是偏小一档；分不清「查看提醒」和「设置提醒」，点错后助手回答对不上；家人绑定弹窗在笔记本大屏上右边被挡住一截（后已修复）。

### 4. 对字号、按钮的感受

特大字号档位有用；希望默认再大一点。按钮大，但手指抖时仍怕点错发送。

### 5. 对语音功能的感受（重点）

非常需要语音。没有示例模板时识别率差；增加「请这样说：血压145/92」后，第二次录入成功。
