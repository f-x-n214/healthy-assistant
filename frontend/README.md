# 前端：银发健康助手对话（免安装版）

这是一个**纯静态前端**对话页 Demo，不依赖 npm/框架，双击即可打开，适合你当前环境里 npm 版本不匹配的情况。

## 如何运行

1. 打开文件：`frontend/index.html`
2. 直接在浏览器里使用对话界面即可

## 内置功能

- 银发友好 UI：大字体、高对比、按钮大
- 快捷按钮：记录用药 / 记录血压 / 健康问答
- 本地 mock：模拟“三级意图识别”
  - 第一层：正则快速命中（如血压 `145/92`、紧急词等）
  - 第二层：简化 AI 层（更宽松的意图判断 + 必要追问）
  - 第三层：安全检查（紧急/处方改药/高血压阈值）

## 下一步（如果你要接后端）

把 `app.js` 里的 `assistantReply()` 替换成调用你自己的接口即可（例如 `/api/chat`），并把第二/三层放到服务端做强约束。
